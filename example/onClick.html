Deferred Plugins Example - Simple Theme - Autolink

This example shows how one could use the Deferred Plugins Plugin to create a page where plugins aren't loaded until they are actually needed by the user. An example of this would be a custom toolbar where some functionality isn't exposed by default but a more advanced version can be enabled by the user. The author of such a toolbar could lower page load time by deferring the loading of plugins until the user actually enabled the more advanced functionality.

On this page the autolink plugin is loaded on demand, try typing a url into the TinyMCE instance below and notice that it's not auto linked. Then click the link below to enable auto linking.
